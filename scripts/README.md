# AI Workforce Intelligence Dashboard

## Project Overview

This project analyzes the impact of Artificial Intelligence on jobs and workforce trends using Python, Machine Learning, and Power BI.

## Dataset

- Total Records: 3000
- Total Features: 20

## Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-Learn
- Git & GitHub

## Key Insights

### Top Job Roles
- Software Developer
- HR Specialist
- Robotics Engineer
- Cybersecurity Analyst

### Highest Paying Jobs
- HR Specialist
- Cloud Architect
- IoT Specialist
- Business Analyst

### Top Hiring Countries
- UAE
- Germany
- Australia
- Pakistan
- Japan

## Machine Learning Model

Random Forest Classifier was used to predict Hiring Trends.

### Target Variable

- Growing
- Stable
- Declining

### Model Accuracy

35%

### Important Features

1. Average Salary USD
2. Job Satisfaction
3. Performance Score
4. AI Replacement Risk
5. Future Demand Score
